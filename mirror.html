Как создать зеркало бота

На этой странице представлена подробная инструкция по созданию зеркальной версии нашего бота. Процесс полностью выполняется через официального Telegram бота @BotFather.

1. Открываем BotFather

В Telegram найдите официального бота BotFather — он отвечает за создание и управление всеми ботами. Нажмите на кнопку ниже, чтобы открыть его:

Открыть BotFather

2. Создаём нового бота (/newbot)

После открытия BotFather отправьте команду /newbot. Он попросит придумать название для вашего зеркального бота. Это просто отображаемое имя — например:
Mirror, Cherny Krest Mirror или любое другое.

3. Выбираем username для бота

Далее BotFather попросит указать username — уникальное имя, по которому можно открыть бота. Оно обязательно должно заканчиваться на bot или _bot. Примеры корректных вариантов:
MirrorBot, mirror_checker_bot, ChernyKrestMirror_bot

4. Получаем токен API

После выбора username BotFather автоматически отправит вам сообщение с API-токеном. Этот токен — ключ, с помощью которого ваше зеркало сможет работать как полноценный бот. Его необходимо бережно хранить и никому не передавать.

Скопируйте токен из сообщения — он понадобится для запуска зеркала.

Готово!

Теперь вы можете вставить этот токен в бота @cherny_krest_bot и зеркало активируется.
